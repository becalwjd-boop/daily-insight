# REPORT_PROMPT

## Daily Insight News Project

Version : 1.0

Status : Official Prompt

Last Updated : 2026-07-03

---

# 목적 (Purpose)

본 문서는 Daily Insight News 프로젝트의 Original Project Report를 생성하기 위한 공식 프롬프트를 정의합니다.

모든 REPORT는 본 프롬프트를 사용하여 생성합니다.

REPORT는 프로젝트의 역사(History)를 보존하기 위한 원본 기록(Evidence)이며,

MASTER.md 및 프로젝트 공식 문서를 작성하기 위한 근거 자료로 사용됩니다.

---

# 사용 방법

1.

개발 채팅이 종료되면

해당 채팅에서 아래 프롬프트를 그대로 입력합니다.

2.

생성된 Original Project Report를 검토합니다.

3.

REPORT_TEMPLATE.md 양식에 맞춰

docs/reports/REPORT_XX.md

파일로 저장합니다.

4.

생성된 REPORT는 이후 수정하지 않습니다.

---

# Official Prompt

(여기에 Version 1.0으로 확정한 REPORT 생성 프롬프트를 수정 없이 그대로 붙여넣습니다.)

---

# 출력 결과

본 프롬프트를 실행하면 다음 항목을 포함한 Original Project Report가 생성되어야 합니다.

1. 프로젝트 개요

2. 새롭게 구현된 기능

3. 기존 기능 변경

4. 해결된 문제

5. 아직 해결되지 않은 문제

6. 기술 부채 (Technical Debt)

7. 프로젝트 방향에 영향을 준 결정

8. Decision Reason

9. 성능 관련 내용

10. Android 관련 내용

11. Archive 관련 내용

12. 개발 원칙

13. docs 반영 대상

14. 채팅 종료 시 프로젝트 상태

15. Related Documents

16. Fact Check

17. 최종 프로젝트 관점 정리

---

# 관리 원칙

본 프롬프트는 프로젝트의 공식 REPORT 생성 프롬프트입니다.

Version이 변경되지 않는 한

프롬프트 내용은 수정하지 않습니다.

새로운 버전이 필요한 경우에는

Version을 증가시키고 변경 이력을 추가합니다.
